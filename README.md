# Issue Tracker

This repository is dedicated to collecting and managing issues, including bug reports, feature requests, and general feedback.

## 📌 Purpose

Instead of scattering issues across multiple places, this repository serves as a **centralized hub** for:

* 🐛 Bug reports
* 💡 Feature requests
* 🗣 Feedback & suggestions
* ❓ Questions (if applicable)

---

## 🚀 How to Use

1. Go to the **Issues** tab
2. Click **New Issue**
3. Choose the appropriate type (Bug / Feature / Feedback)
4. Provide as much detail as possible

---

## 🧾 Issue Guidelines

To help us resolve issues faster, please include:

### 🐛 For Bug Reports

* Clear description of the problem
* Steps to reproduce
* Expected behavior
* Screenshots / logs (if possible)
* Environment (OS, version, etc.)

### 💡 For Feature Requests

* What you want to achieve
* Why it is needed
* Possible implementation (optional)

---

## 🏷 Labels

Issues may be categorized using labels such as:

* `bug`
* `enhancement`
* `question`
* `documentation`
* `invalid`
* `wontfix`

---

## 🙏 Notes

* Please search existing issues before creating a new one
* Be respectful and constructive
* Keep reports clear and concise

---
